• 1.两种模块化的写法

    • amd 异步模块加载

      先行原则

      基本语法

      define(id, deps, factory);
      define(deps, factory);

      范例:

      define("id", ["dep1","dep2","dep3"], function(require, exports, module, dep1, dep2, dep3){{ }});

    • cmd 通用模块定义

      就近原则

      基本语法

      define(id, factory);

      范例:

      define("id", function(req, exp, mod){{ var dep1 = req("dep1"); var dep2 = req("dep2"); var dep3 = req("dep3"); }});

  • 2.factory参数

    • require(req) 请求模块
    • exports(exp) 输出对象
    • module(mod) 模块本身

  • 3.config配置

    • 1.写法: {literal}seekjs.config({ns:{-},path:{-},alias:{-},packages:{-}});{/literal}
    • 2.ns 命名空间 如: req("mod.Dream");
    • 3.path 短路径 如: req("ad/160101/home");
    • 4.alias 别名 如: req("service");
    • 5.packages 包 如: req("echarts")/req("cheachs/line");
